⚠️ 尚未配置 Firebase — 当前为本地单机模式，刷新后数据丢失，多人无法共享。
  1. 打开 console.firebase.google.com，新建项目（可关闭 Google Analytics）
  2. 左侧菜单 → 构建 → Realtime Database → 创建数据库，选离你最近的区域，以"测试模式"启动
  3. 在数据库页面顶部复制 databaseURL（格式：https://xxx-default-rtdb.区域.firebasedatabase.app）
  4. 项目设置（齿轮图标）→ 常规 → 你的应用 → 添加 Web 应用，复制 firebaseConfig 对象
  5. 用编辑器打开 index.html，将文件底部 firebaseConfig 占位内容替换为你的配置，重新推送到 GitHub
